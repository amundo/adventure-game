Utility components

    1. <data-viewer> - displays generic data.component-index.css

    Game components

    1. <game-manager> - Entry point of the game, initializes everything else.
    2. <game-ui> - Listens for user actions and relates them to the map.
    3. <game-map> - The map of the world. Loads and renders map data where instructed by GameUi.
    4. <game-controls> - The controls that the user interacts with. Handles listening for key events, mouse events, etc events.
    5. <game-cell> - Each cell in the map. Has an x, y property which places it in the grid. Tracks what is on it and terrain type.
    6. <enemy-character> - represents an enemy, placed on the board by the GameManager.
    7. <player-character> - represents the player, placed on the board at the beginning of the game.
    8. <game-item> - Represents an item, placed on the board by the GameManager.
