Adventure game framework

Game components

  1. <game-manager> Entry point of the game, initializes everything else.
  2. <game-ui> Listens for user actions and relates them to the map.
  3. <game-map> The map of the world. Loads and renders map data where instructed by GameUi.
  4. <manage-map> Handles file and metadata operations for a game map.
  5. <game-controls> The controls that the user interacts with. Handles listening for key events, mouse events, etc events.
  6. <game-cell> Each cell in the map. Has an x, y property which places it in the grid. Tracks what is on it and terrain type.
  7. <enemy-character> represents an enemy, placed on the board by the GameManager.
  8. <player-character> represents the player, placed on the board at the beginning of the game.
  9. <game-item> Represents an item, placed on the board by the GameManager.

Utility components

  1. <data-viewer> displays generic data.
